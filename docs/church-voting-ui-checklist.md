# 教會投票系統 UI 欄位清單（逐頁）

本文件可直接交給 UI/前端設計師與 PM 做畫面規格確認。

## A. 前台（會友）

## A1. QR 進入後驗證頁

- 顯示欄位
  - 教會名稱 / 系統名稱
  - 簡短隱私說明（無記名、一次性投票）
  - 錯誤訊息區（token 無效、過期、已使用、非投票時段）
- 行為
  - 自動讀取 URL token
  - 驗證成功後自動跳轉投票頁

## A2. 投票主頁

- 固定資訊
  - 選舉名稱（如 2026 長老選舉）
  - 應選名額（N 名）
  - 投票截止時間
  - 已選數量（即時）
- 候選人卡片
  - 候選人姓名（必要）
  - 照片（選配）
  - 簡介（選配）
  - 勾選框/切換按鈕
- 按鈕
  - 清除已選（選配）
  - 確認送出
- 驗證規則
  - 已選數量 <= 應選名額
  - 允許少選（0 ~ N）
  - 超選時禁用送出並顯示提示

## A3. 送出確認彈窗

- 內容
  - 已選候選人名單（按顯示順序）
  - 提示文字：送出後不可修改
- 按鈕
  - 返回修改
  - 確認送出

## A4. 投票成功頁

- 顯示
  - 成功訊息
  - 提交時間
  - 客服/聯絡資訊（選配）
- 行為
  - 關閉頁面後不可再次投票

---

## B. 後台（選務管理）

## B1. 管理員登入頁

- 欄位
  - 帳號
  - 密碼
  - MFA 驗證碼（若啟用）
- 行為
  - 登入成功導向儀表板
  - 失敗顯示具體原因（不洩漏敏感訊息）

## B2. 儀表板

- 統計卡片
  - 進行中選舉數
  - 總投票率
  - 已使用 token / 未使用 token
  - 最近管理操作（稽核摘要）
- 列表
  - 選舉狀態表（草稿/進行中/已結束）

## B3. 選舉設定頁

- 欄位
  - 選舉名稱
  - 類型（長老/執事）
  - 應選名額
  - 開始時間
  - 結束時間
  - 狀態（草稿）
- 按鈕
  - 儲存草稿
  - 啟用選舉（或排程）
  - 結束投票（進行中才顯示）
- 驗證
  - 開始時間 < 結束時間
  - 名額 > 0

## B4. 候選人管理頁

- 表格欄位
  - 排序
  - 姓名
  - 簡介
  - 啟用狀態
  - 操作（編輯/停用/刪除）
- 編輯抽屜/彈窗
  - 姓名（必要）
  - 照片上傳（選配）
  - 簡介（選配）
  - 排序（必要）

## B5. 會友管理頁

- 匯入區
  - CSV 上傳
  - 欄位對應預覽（memberNo, name, phone/email）
  - 匯入錯誤報告下載
- 表格欄位
  - 會員編號
  - 姓名
  - 聯絡方式
  - 狀態（啟用/停用）

## B6. 金鑰/QR 管理頁

- 篩選
  - 選舉
  - token 狀態（未使用/已使用/已作廢）
- 表格欄位
  - 會員編號
  - 姓名
  - token 狀態
  - 發放時間
  - 使用時間
  - 操作（作廢重發）
- 按鈕
  - 一鍵產生 token
  - 匯出 QR（PDF）

## B7. 投票控制頁

- 顯示
  - 現在時間
  - 投票開始/結束時間
  - 目前狀態
- 操作
  - 立即開始
  - 立即結束（需要二次確認）

## B8. 開票與報表頁

- 顯示
  - 候選人得票數
  - 投票率
  - 無效票數（若有設計）
- 匯出
  - CSV
  - PDF
- 限制
  - 建議僅在選舉結束後可見完整結果

## B9. 稽核紀錄頁

- 欄位
  - 時間
  - 管理員
  - 操作類型（建立選舉/改名額/手動結束等）
  - 內容摘要
- 功能
  - 依日期與操作類型篩選
  - 匯出紀錄

---

## C. 全站通用元件

- 對話框：危險操作需二次確認
- Toast：成功/失敗訊息
- 空狀態：尚無資料時提示下一步操作
- 載入狀態：避免重複送出
- 權限提示：未授權頁面顯示禁止訪問
